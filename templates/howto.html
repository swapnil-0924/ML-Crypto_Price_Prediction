How to Use CryptoPredict

A step-by-step guide to making accurate predictions

1

Select Cryptocurrency

Enter any valid cryptocurrency ticker symbol (e.g., BTC-USD, ETH-USD) in the input field.

2

Choose Prediction Window

Select the number of days (1-365) for which you want to forecast the price.

Understanding the Results

  • Forecast Plot: Shows predicted prices with confidence intervals.
  • Predictions Table: Displays detailed price predictions for each day.
  • Metrics: RMSE, MAE, and MAPE indicate prediction accuracy.
